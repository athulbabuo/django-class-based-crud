# ⚙️ Django Class-Based Views CRUD Project

## 🔹 Overview

This project demonstrates how to implement CRUD (Create, Read, Update, Delete) operations using Django’s Class-Based Views (CBV).
It highlights the use of Django generic views to build clean, reusable, and maintainable web applications.

---

## 🚀 Features

* ➕ Create new records
* 📋 View list of records
* ✏️ Update existing records
* ❌ Delete records
* ♻️ Reusable and clean code using CBV
* 🧩 Structured project architecture

---

## ⚙️ Installation & Setup

1. Clone the repository

```id="3qwd2l"
git clone https://github.com/athulbabuo/django-class-based-crud.git
```

2. Navigate to the project folder

```id="h82k1p"
cd django-class-based-crud
```

3. Create virtual environment

```id="x93k2v"
python -m venv venv
```

4. Activate environment

* Windows:

```id="l29c8s"
venv\Scripts\activate
```

* Mac/Linux:

```id="z91l2a"
source venv/bin/activate
```

5. Install dependencies

```id="u72k1p"
pip install -r requirements.txt
```

6. Apply migrations

```id="m38l2s"
python manage.py migrate
```

7. Run server

```id="q91k3p"
python manage.py runserver
```

8. Open in browser

```id="v72l1s"
http://127.0.0.1:8000/
```
